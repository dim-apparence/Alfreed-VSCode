# Koby

## 🧐 What is it ?

Easily create some Flutter page pre-filled like:
- MVVM [alfreed](https://pub.dev/packages/alfreed) page
- MVVM stateful page
- Service
- Loader
  
More are coming next... 🤩

## 🎯 How it works ?

Just right click on the folder you want 💪
